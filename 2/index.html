Respuestas:

  1. El loop de javascript se ejecuta casi de inmediato, por lo tanto si hay una función como "setTimeout" lo que pasará es que el bucle terminará antes de la ejecución de código dentro del "setTimeout", y dará como resultado en consola el último valor (repetido varias veces) registrado dentro del bucle.

  2. He creado una función "counter" que se encarga de llamarse a sí misma recursivamente una vez termine el tiempo del "setTimeout", parametrizado de manera que pueda personalizarse de forma sencilla.
